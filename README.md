# Южные Раскаты — лендинг

Статический одностраничный сайт (HTML/CSS/JS, без сборки и зависимостей) для продажи земельных участков в посёлке «Южные Раскаты» под Красноярском.

**Сайт опубликован:** https://yuzhnye-raskaty.netlify.app

## Как опубликовать обновления

Репозиторий на GitHub (`fiarovanti-byte/yuzhnye-raskaty`) хранит код, а хостинг сайта — на Netlify. Автосвязки между ними нет, поэтому после правок нужно вручную задеплоить:

```bash
npx netlify-cli deploy --dir=. --prod --site=ed192c79-1c03-4c97-bc46-2e046eb02e5e
```

(Потребуется быть залогиненным в Netlify CLI: `npx netlify-cli login`.)

## Структура

```
index.html        — вся разметка страницы
css/styles.css     — дизайн-система и стили (светлая/тёмная тема через prefers-color-scheme)
js/main.js         — мобильное меню, аккордеон FAQ, скролл-анимации появления секций
assets/images/     — фотографии (сгенерированы, см. ниже)
```

Иконки подключены с CDN (jsDelivr, пакет `@phosphor-icons/web`). Шрифт — системный стек (`-apple-system, Segoe UI, Roboto...`), без загрузки веб-шрифтов: страница грузится быстро и одинаково хорошо выглядит на Windows/macOS.

## Как посмотреть сайт локально

Просто открыть `index.html` в браузере. Либо поднять локальный сервер, например:

```bash
npx serve .
```

## Технические заметки

- Тёмная/светлая тема переключается автоматически по системным настройкам пользователя (`prefers-color-scheme`).
- Анимации появления секций отключаются автоматически, если у пользователя включён режим уменьшенного движения (`prefers-reduced-motion`).
- Сайт адаптивен: мобильная навигация — через кнопку-гамбургер, все секции перестраиваются в одну колонку на узких экранах.
